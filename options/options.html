Featherwise

Access token

Paste your token from readwise.io/access_token. It is stored locally in this extension and sent only to readwise.io.

Full-page HTML capture

When off, saving sends only the page URL and Readwise fetches the article. Turn this on to send the page's own HTML instead — needed for paywalled or logged-in pages. Enabling it grants this extension the scripting permission, which it holds only while this is on.

Keyboard shortcut

The default is Ctrl/Cmd+Shift+S to open the popup. Change it at chrome://extensions/shortcuts.
